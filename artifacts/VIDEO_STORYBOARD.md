# Demo Video Storyboard (90-120 seconds)

## Goal

Show product and platform thinking, not just UI clicks.

## Recording Specs

- Length: 90-120 seconds
- Format: `demo.mp4`
- Aspect: 16:9 (1920x1080 preferred)
- Audio: optional; subtitles are enough

## Scene Plan

1. 0:00-0:08 | Context
- Visual: title card + one sentence value proposition
- Subtitle: "Family task platform with closed-loop execution and incentives"

2. 0:08-0:28 | Parent Task Creation
- Visual: template-based task creation, assignment, publish
- Subtitle: "Fast task publishing with reusable templates and controlled assignment"

3. 0:28-0:45 | Child Execution
- Visual: child claims/completes task
- Subtitle: "Execution state is explicit and traceable"

4. 0:45-1:03 | Review and Points
- Visual: parent approves/rejects submission, points update
- Subtitle: "Approval path drives trusted points integrity"

5. 1:03-1:18 | Rewards + Engagement
- Visual: reward redemption and leaderboard/rivals
- Subtitle: "Behavior loop reinforced through incentives"

6. 1:18-1:30 | Security + Platform Close
- Visual: parent PIN/password gate + architecture image
- Subtitle: "Step-up verification and platform boundaries support safe scale"

## Optional Voiceover Script

- "This is Kingby, a task collaboration platform for families."
- "Parents publish tasks quickly using templates and assignment controls."
- "Children execute and submit tasks through a clear workflow."
- "Parents review submissions and points update through controlled state transitions."
- "Rewards and ranking features reinforce consistent completion behavior."
- "Security-sensitive transitions use password or PIN verification."

## Editing Checklist

- Add callout labels for each core workflow
- Keep transitions under 300ms; no flashy effects
- Ensure text remains readable on laptop screens
- Export as `artifacts/demo.mp4`
