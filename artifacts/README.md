# Artifacts

Public-safe portfolio assets used by this repository.

## Files

- `SCREENSHOT_CHECKLIST.md` - required and optional screenshots to capture
- `VIDEO_STORYBOARD.md` - 90-120 second recording plan
- `COVER_TEMPLATE.svg` - editable visual cover template (export to PNG)
- `ASSET_PRODUCTION_RUNBOOK.md` - step-by-step production sequence
- `screenshots/` - finalized screenshots
- `demo.mp4` - finalized walkthrough video (add after recording)
- `architecture.png` - architecture diagram (add/update)

## Security Checklist

- No API keys, tokens, or secrets in screenshots/video
- No private customer data or personal identifiers
- No internal-only admin URLs or infrastructure details
